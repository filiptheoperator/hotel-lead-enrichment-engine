# Hotel Lead Enrichment Engine OS — V3 Starter Kit

## Účel balíčka
Tento balíček je pripravený na okamžitý štart nového projektu pre:
- automatizáciu sourcing → cleaning → enrichment → e-mail draft → CRM sync workflowu,
- zrýchlenie práce pred ostrým outreachom,
- zachovanie vysokej kvality hotelového enrichmentu,
- vytvorenie stabilného build systému pre ďalšie iterácie.

## Čo je v balíčku
1. projektové inštrukcie,
2. master SOP o biznise a workflowe,
3. implementation plan,
4. repo štruktúra,
5. GitHub backup SOP,
6. starter README,
7. `.env.example`,
8. `.gitignore`,
9. `requirements.txt`,
10. YAML config súbory,
11. štartovacie prompty,
12. build prompty pre Codex.

## Pravidlo práce s týmto projektom
- Odpovedať iba po slovensky.
- Odpovede držať čo najkratšie.
- Keď ide o technický krok, viesť za ruku presne a po krokoch.
- Technické odpovede deliť na:
  - VS Code
  - Codex web
  - Validácia
- Po dôležitých krokoch pripomínať GitHub commit a push.
