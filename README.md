# Hotel Lead Enrichment Engine OS

## Project
Hotel Lead Enrichment Engine OS

## Purpose
Automatizovať workflow:
- raw hotel leads
- cleaning and ranking
- hotel enrichment
- personalized email draft generation
- ClickUp-ready exports

## Core goal
Znížiť ručnú prácu a zachovať vysokú kvalitu enrichmentu.

## Main folders
- `configs/` = project configuration
- `data/` = raw and processed data
- `outputs/` = final exports
- `prompts/` = prompt assets
- `src/` = application code
- `docs/` = documentation

## Main outputs
- ranked accounts
- hotel enrichment master
- per-hotel markdown briefs
- email drafts
- ClickUp import files

## Working rule
Najprv build stability, potom automation.
