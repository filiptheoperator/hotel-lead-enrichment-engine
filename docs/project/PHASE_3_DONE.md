# Fáza 3 - Definition of Done

## Stav

Fáza 3 je uzatvorená ako `operator-ready workflow`.

## Potvrdené hotové

- output contract je uzamknutý
- operator batch checklist existuje
- run manifest existuje
- ClickUp gate a High-only gate existujú
- batch readiness score a readiness band existujú
- readiness explanation artifact existuje
- operator decision summary existuje v TXT aj CSV
- High-only operator summary existuje v TXT aj CSV
- full aj High-only ClickUp export existujú
- full aj High-only dry run sample existujú
- archive workflow a cleanup pravidlá existujú
- ClickUp API mapping preview, validation a payload diff existujú
- Make-ready handoff spec existuje

## Známe limity

- živý ClickUp import rehearsal je stále neoverený
- finálne custom field IDs nie sú potvrdené
- Make orchestration ešte nie je implementovaná

## Výstup Fázy 3

- operátor má 1-screen aj detailné artifacty pre rozhodnutie
- batch je pripravený na riadený prechod do integračnej fázy
