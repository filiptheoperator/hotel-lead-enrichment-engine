# Make ClickUp Task Field Mapping

## Cieľ

Potvrdiť, ktoré ClickUp polia má Make vetva používať a ktoré nemá meniť.

## Povinné polia

- `name` -> `Task name`
- `description` -> `Description content`
- `status` -> `Status`
- `priority` -> `Priority`
- `custom.hotel_name` -> `Hotel name`
- `custom.city` -> `City`
- `custom.priority_score` -> `Priority score`
- `custom.subject_line` -> `Subject line`
- `custom.source_file` -> `Source file`

## Voliteľné polia

- `custom.contact_phone` -> `Contact phone`
- `custom.contact_website` -> `Contact website`

## Make vetva nemá meniť

- outreach testing logiku
- enrichment scoring logiku
- QA decision logiku
