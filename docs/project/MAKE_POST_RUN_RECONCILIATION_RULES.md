# Make Post Run Reconciliation Rules

## Skontrolovať

1. response capture
2. decision summary
3. status board
4. archive cross-check
5. ClickUp output alebo stop vetvu

## Pass

- response je úspešná
- output reference sedí
- nevznikol neočakávaný partial write

## Fail

- response chýba
- output reference nesedí
- klasifikácia je `client_error` alebo `cloudflare_block`
