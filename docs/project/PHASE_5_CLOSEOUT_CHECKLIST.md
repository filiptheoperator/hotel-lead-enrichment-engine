# Phase 5 Closeout Checklist

## Interné closeout podmienky

1. orchestration runner je stabilný
2. payload contract je uzamknutý
3. retry contract je uzamknutý
4. operator handoff existuje
5. Make resume point existuje
6. success a failure output contract existujú
7. live Make retest má pripravený presný resume point

## Stav

- interná časť Fázy 5: `DONE`
- externý Make network blocker: `OPEN`

## Phase 6 vstup je povolený keď

- interná časť Fázy 5 je hotová
- otvorené externé blockery sú explicitne oddelené
